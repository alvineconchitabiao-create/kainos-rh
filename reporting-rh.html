Reporting RH Automatique
Type : Période : Données lues depuis les outils connectés
🔗 Sources de données connectées
Source principale : Le Registre du Personnel (elitereg_v1) fournit toutes les données d'effectif : actifs, entrées, sorties, anciennetés, masse salariale, pyramide des anciennetés, mouvements.
Les autres outils enrichissent le rapport avec les données opérationnelles (paie, absences, HS, congés, prêts).
Tous les fichiers doivent être ouverts dans le même navigateur pour que les données soient disponibles.
